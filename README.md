# Expense Tracker

This is a simple Expense Tracker project developed using Python. The main purpose of this project is to track daily expenses and calculate the total amount spent. Users can add expenses, view all expenses, calculate total spending, and check the highest expense.

This project helped me improve my understanding of Python programming concepts such as loops, dictionaries, lists, conditions, functions, and user input handling. It also helped me understand how real-world finance tracking applications work.

## Features

* Add New Expense
* View All Expenses
* Calculate Total Expenses
* Find Highest Expense
* Menu Driven System
* User Friendly Console Interface

## Technologies Used

* Python

## Concepts Used

* Variables
* Loops
* Conditions
* Lists
* Dictionaries
* Functions

The purpose of this project is to improve problem-solving skills and build real-world programming logic using Python.
